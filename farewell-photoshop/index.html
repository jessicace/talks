Blend Modes

Support

Global: 54.73% + 9.78% = 64.5%

Not supported in Opera Mini, Internet Explorer, or Edge.

mix-blend-mode on SVG elements is currently only supported in Firefox.

Under consideration in Edge, but has a low priority.

In Safari, hue, saturation, color, and luminosity are not supported :(

Polyfills using the canvas are available for background-blend-mode.

isolation is only supported in Firefox and Chrome.

Considerations

Consistency

Don't expect the same colours on every device OR browser!

Accessibility

Ensure contrast on text.

Make sure your design still works!

Stacking Context

mix-blend-mode creates a new stacking context.

Only solution is to figure out how stacking contexts work.
